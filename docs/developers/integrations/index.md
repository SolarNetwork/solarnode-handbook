# Integration APIs

SolarNode offers different APIs for integrating with the runtime.

| Integration | Description |
|:------------|:------------|
| [REST](./rest/index.md) (HTTP) | A REST API over HTTP using primarily JSON messages. |
| [WebSocket](./websocket/index.md) (HTTP) | An HTTP WebSocket API for real-time JSON messages using the STOMP sub-protocol. |
| [STOMP](./stomp/index.md) (TCP) | A STOMP TCP connection using JSON messages for bidirectional real-time system management. |
